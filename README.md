
# Global Superstore Business Performance Analysis
## Project Overview

This project presents an end-to-end business analysis of the Global Superstore dataset using Python. The objective is to identify the main factors driving sales, profitability, customer performance, product success, and operational efficiency across multiple international markets.

The analysis follows a business-oriented workflow, starting with data cleaning and preprocessing before answering twenty business questions through statistical analysis and data visualization.

## Objectives
Clean and prepare the dataset for analysis.
Evaluate sales and profit across countries, markets, and regions.
Identify top-performing and underperforming products.
Analyze customer purchasing behavior.
Examine shipping performance and operational efficiency.
Generate actionable business recommendations supported by data.
## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
## Business Questions

The project answers twenty business-oriented questions related to:

Data quality
Sales performance
Profitability
Regional analysis
Customer segmentation
Product performance
Shipping efficiency
Order priorities
## Key Insights
Sales are concentrated in a limited number of countries.
Profitability varies significantly across regions.
Several products consistently generate financial losses.
Customer value is highly uneven.
Standard Class shipping represents the company's primary logistics channel.
Shipping time has minimal influence on profitability.
## Business Value

The findings can support management decisions regarding pricing strategies, inventory optimization, logistics planning, customer relationship management, and market expansion.
